# Taigaschool Eco Hotel - Investment One-Pager

Professional investment presentation website for Taigaschool Eco Hotel property valuation.

## Features

- **Bilingual**: English & Finnish language support
- **Responsive Design**: Optimized for all devices
- **Key Metrics**: Revenue, NOI, Valuation, Financing
- **Interactive Elements**: Language toggle, scroll animations
- **Professional Design**: Corporate/institutional aesthetic

## Deployment

Deployed on Vercel: [https://taigaschool-onepager.vercel.app](https://taigaschool-onepager.vercel.app)

## Local Development

```bash
# Serve locally
npx serve .

# Or simply open index.html in browser
```
